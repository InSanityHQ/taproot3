\section{Emergent Properties}
\label{sec:org6736510}
\begin{itemize}
\item An emergent property is a property within a unit that creates a
structure when coupled with other units with similar properties.i

\begin{itemize}
\item This self-organization based on a property or set of properties is
called an emergent property.
\end{itemize}
\end{itemize}

\subsection{Bird Example}
\label{sec:org3411484}
\begin{itemize}
\item Bird's flying in a V is a great example of what an emergent property
is.

\begin{itemize}
\item This is because it demonstrates a multi-level organization system.
\end{itemize}
\end{itemize}
