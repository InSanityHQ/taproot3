\section{The Ottomans' Successes in the 1500s}
\label{sec:org42f412d}
\textbf{3 main factors of growth}

\begin{itemize}
\item Original leader was quite shrewd

\item Strategic link between Europe + Asia at the Daranelles strait

\item Creation of army combining Trukish techniques + high tech gunpowder

\item Eastern frontier => where Safavid was

\item Southern frontier => red sea
\end{itemize}

\subsection{The Early Rise}
\label{sec:org2cd2fb6}
The country started with a mix of three cultures

\begin{itemize}
\item Turkish Nomads
\item Muslim Warriors
\item Christian Converts
\end{itemize}

In 1453, Constantinople was conquered by the Ottomans, which ended
Byzantine rule + gave a pretty good capital

\subsection{The Ottomans' Battlings}
\label{sec:orga9fe0db}
\begin{itemize}
\item Conquered Safavid Empire => 1514
\item Seized Damascus in the year 1516
\item Conqured Egypt 1517
\item Worked around, and all the way to the Black Sea --- attacking Arabia
along the way
\item Overran Hungary (although, Northern Hungary was kept, and so was the
HRE) the same year
\item Seeked control of the Mediterranean
\item (Unsuccessfullyl) tried to fight Venice

\begin{itemize}
\item Initial fight => reduced millitary power + tribunage
\item Finally fell another century later, but due to unrelated reasons
\end{itemize}

\item Fought the Porchugueses, too!

\begin{itemize}
\item Tried to deter porchuguese threats
\item Successfully prevented them from closing in, but ultimately did not
conquer into Porchigual
\end{itemize}
