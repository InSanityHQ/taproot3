Functions vs Maps

Technically, maps don't need to be well defined, meaning in a map one input might be allowed to go to multiple outputs. However, the linear maps that we are looking at will always be well defined. Maps give you a notion of "space" where as a function implies that the output depends on the input. Both are true, but each one gives a intuition for part of the picture.

Proof by Contradictioncareful
