Bijective Functions

Written by Exr0n
<>#definition bijectivedef

A map is bijective if it is both injective and bijective.

Results

If a map $T : V \to W$ is bijective, then $\text{dim }V = \text{dim }W$ by Axler3.23 and Axler3.24.
