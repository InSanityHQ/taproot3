\section{Types of Discontinuity}
\label{sec:org10dfda0}
\subsection{Infinite Discontinuity (verticle asymtote)}
\label{sec:orga90bb46}
\textbf{Double Sided Limit}: does not exist \textbf{Function}: not defined

\(f(x)=\frac{x^2+2}{x^2-4}\ where\ x=2\)

\subsection{Jump Discontinuity (gap)}
\label{sec:org62ae0fa}
\textbf{Double Sided Limit}: does not exist \textbf{Function}: defined

"a ceiling function" \(f(x) = \left \lceil{x}\right \rceil\ where\ x=3\)

\subsection{Point Discountinuity (hole)}
\label{sec:orga27c04b}
\textbf{Double Sided Limit}: exists \textbf{Function}: not defined

\(f(x)=\frac{x^6-1}{x^{10}-1}\ where\ x=1\)

\noindent\rule{\textwidth}{0.5pt}

Division by 0/infinite discontinutiy => undefined

Hole => not defined

Sq. Root Functions => not defined when x=\{negative number\}
