\section{Derive the kinematic equations for constant acceleration}
\label{sec:org7e11aa4}

\[\begin{aligned}
  v &= \int a dt &= at + C_v\\
  x &= \int v dt = \int (at+C_v) dt &= \frac{1}{2}at^2 + C_v t + C_x
  \end{aligned}\]

Letting \(x_0 = C_x\) and \(v_0 = C_v\),
\[\begin{aligned}
  x = x_0+v_0t+\frac{1}{2}a_0t^2
  \end{aligned}\]
