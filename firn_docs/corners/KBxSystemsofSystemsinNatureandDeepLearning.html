directed graphs as meta-abstraction

for topology, an arrow represents a map between two objects. ofc, there are essentially inf possible maps - a map is a continuous map where topology is preserved when mapping specific points between two objects - the "verbs" are the maps - between any two nodes, there is a bundle of maps going to and from every node - + reflective loops, maps to themselves - instead, we represent a bundle in a direction as a single arrow - making this directed graph for every possible topology is acategory
