Random Fields

A random field is a stochastic process within Euclidean space and with dimensionality > 1. What's a stochastic process? Note that $T$ is a traditional Euclidean space like a cube.DEFINITION A stochastic process $f$ over $T$ is a collection of random variables $\{f(t) : t \in T\}$.DEFINITION If $T$ is a set of dimension $N$, and the random variables $f(t)$ are all vector valued of dimension $d$, then we call the vector valued random field $f$ a $(N, d)$ random field.

An intuitive example of a random field is the ocean surface.
