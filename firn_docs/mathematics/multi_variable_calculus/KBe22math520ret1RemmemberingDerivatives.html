checking partial derivatives

Is the following possible? \[\begin{aligned} \frac{\partial}{\partial x} f = 2x + 3y \\ \frac{\partial}{\partial y} f = 4x + 6y \\ \end{aligned}\]

No, because if you integrate them, you have completely different functions. \[\begin{aligned} \int (2x + 3y) dx &= x^2 + 3xy + C\\ \neq \int (4x + 6y) dy &= 3y^2 + 4xy + C \end{aligned}\]

Upon checking my work with Ian, I realize that doing the mixed partial is probably cleaner because then there's no ambiguous $+C$ term.
