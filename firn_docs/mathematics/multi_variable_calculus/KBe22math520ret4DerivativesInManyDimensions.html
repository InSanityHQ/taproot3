surface slope by parameterization

the slope

Let $g(t) = \langle x + t \cos \theta , y + t \sin \theta \rangle$ be the parameterization s.t. the slope at $(f \circ g)(0)$ is the slope that we want. \[\begin{aligned} \frac{d}{dt} f \circ g \Bigr|_0 \end{aligned}\]

the angle

We want to find the $\theta$ where the slope is maximal, so we can set the derivative to zero

\[\begin{aligned} \frac{d}{d\theta} \left( \frac{d}{dt} f \circ g \Bigr|_0 \right) = 0 \end{aligned}\]
