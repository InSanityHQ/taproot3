2 Templates

Org Roam capture supports templates (although they are an abuse of the capture system).

Example config: 

("d" "default" plain (function org-roam--capture-get-point)
     "%?"
     :file-name "%<%Y%m%d%H%M%S>-${slug}"
     :head "#+title: ${title}\n"
     :unnarrowed t)

Note that "d" ("default") is the default org roam will select, and parameters like :file-name, :head and :unnarrowed control the respective properties of the file.

More detailed config info can be found here.

3 References

Org Roam notes can contain references to websites or papers (via org-ref). The methods to reference either type of source are shown below. 

#+title: Google
#+roam_key: https://www.google.com/

#+title: Neural Ordinary Differential Equations
#+roam_key: cite:chen18_neural_ordin_differ_equat
