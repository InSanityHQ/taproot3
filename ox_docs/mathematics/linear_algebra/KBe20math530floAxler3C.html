    2 Axler3.32 #def matrix of a linear map, \(\mathcal M(T)\)   def

    Suppose \(T \in \mathcal L(V, W)\) and \(v_1, \ldots, v_n\) is a basis of \(V\) and \(w_1, \ldots, w_m\) is a basis of \(W\). The matrix of \(T\) with respect to these bases is the \(m\times n\) matrix \(\mathcal M\left(\mathcal T, \left(v_1, \ldots, v_n\right), \left(w_1, \ldots, w_m\right)\right)\) whose entries \(A_{j,k}\) are defined by \[Tv_k = A_{1, k}w_1 + \cdots + A_{m, k}w_m\].

    Note that for each output \(Tv_k\) is a linear combination of a column.
